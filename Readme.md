# 💰 Expense Tracker (Flask + SQLite)

A simple Expense Tracker web app built using Flask and SQLite.  
It allows users to manage daily expenses with useful summaries.

---

## Features

-  Add new expenses
-  Edit existing expenses
-  Delete expenses
-  Filter by category
-  Total expense calculation
-  Category-wise summary
-  Monthly summary

---

##🛠 Tech Stack

- Python (Flask)
- SQLite (Database)
- HTML + CSS (Frontend)

## How to Run 

1. Clone the repo:
    git clone <your-repo-link>
    cd Expense_Tracker


2. Install dependencies:
    pip install flask

3. Run the app:
    python app.py

4. Open in browser:
    http://127.0.0.1:8000
